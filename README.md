# Thesis Analysis Repository

This repository contains the notebooks and datasets used for the analysis in my Master's thesis.

## How to run the analysis

1. Download or clone this repository.
2. Open `thesis_notebook.ipynb` in Google Colab or Jupyter Notebook.
3. Run the notebook from top to bottom.

The notebook uses relative file paths, so the folder structure should not be changed.

## Folder structure

- `thesis_notebook1(Source.ipynb`: main analysis notebook
- `thesis_notebook 2.ipynb`: main analysis notebook
- `thesis_notebook 3.ipynb`: main analysis notebook
- `data/`: datasets used in the analysis
